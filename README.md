# WESA_AI
# 🚀 NEXTGEN WESA AI

AI-Powered Website Intelligence & Business Analysis Platform

## 📌 Overview

NEXTGEN WESA AI is an intelligent website analysis platform that helps businesses, consultants, and sales teams extract valuable insights from company websites.

The platform automatically analyzes websites, identifies business information, extracts contact details, detects technologies, generates AI-driven business insights, recommends digital transformation opportunities, and creates implementation plans using Google's Gemini AI.

## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/nextgen-wesa-ai.git
cd nextgen-wesa-ai
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Configure Gemini API Key

Create a `.env` file in the project root directory.

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

Install dotenv support:

```bash
pip install python-dotenv
```

---

## ▶️ Run Application

```bash
streamlit run mainn.py
```

---

## 📈 Workflow

1. Enter company website URL.
2. Click **Analyze Website**.
3. System scrapes website content.
4. AI generates business intelligence.
5. Recommended solutions are produced.
6. Competitor comparison can be performed.
7. PDF report can be downloaded.
